# 🐶 Bucky & Loki — Il Mondo di Coda

Un cozy game per telefono (stile Animal Crossing / Pokopia) con protagonisti **Bucky** e **Loki**, due piccoli Yorkshire.
Esplora il villaggio di Coda, aiuta gli abitanti, raccogli risorse, completa missioni e scopri il mistero dell'**osso d'oro** ✨.

## ▶ Gioca ora
Apri il link di **GitHub Pages** dal telefono o dal browser:
👉 https://pikupfood.github.io/bucky-loki-world/

## 🎮 Comandi
- **Telefono**: joystick in basso a sinistra 🕹️ · pulsante arancione a destra per interagire · 🎒 per lo zaino
- **PC**: frecce / WASD per muoverti · **Spazio** o **E** per interagire

## ✨ Cosa c'è dentro (prototipo)
- Selezione del personaggio (Bucky 🎀 rosso / Loki 🎀 blu) — l'altro ti segue come compagno
- Mondo esplorabile con case, stagno, alberi, cespugli, cuccia e pozzo
- 3 abitanti con dialoghi e una catena di missioni con storia
- Raccolta risorse (ossa, fiori, bacche, bastoncini)
- Zaino / inventario · ciclo dei giorni · salvataggio automatico
